#  LeadFlow - Lead Management System

LeadFlow is a beginner-friendly Full Stack Lead Management System developed using Node.js, Express.js and MySQL.

It allows users to manage business leads efficiently through a clean dashboard with CRUD functionality.


##  Project Preview

Dashboard
- Total Leads
- New Leads
- Contacted Leads
- Qualified Leads
- Closed Leads

Features
- Add Lead
- Edit Lead
- Delete Lead
- Search Lead
- Filter by Status
- Responsive Design

##  Installation

Clone Repository

```

git clone https://github.com/yourusername/LeadFlow.git

```

Move into project

```

cd LeadFlow

```

Install packages

```

npm install

```

Update database password

```

config/db.js

```

Run SQL

```

leadflow.sql

```

Start Server

```

npm run dev

```

Open

```

http://localhost:3000

```

##  API Endpoints

| Method | Endpoint | Description |
|----------|----------------|----------------|
| GET | /api/leads | Get all leads |
| GET | /api/leads/:id | Get single lead |
| POST | /api/leads | Add lead |
| PUT | /api/leads/:id | Update lead |
| DELETE | /api/leads/:id | Delete lead |
